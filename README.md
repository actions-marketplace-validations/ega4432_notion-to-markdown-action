# notion-to-markdown-action

## Environment

- Node.js v16.16.0
- npm v8.11.0

## Setup

Install the dependencies

```bash
$ npm install
```

Build the typescript and package it for distribution

```bash
$ npm run build && npm run package
```

Run the tests :heavy_check_mark:

```bash
$ npm test
```
